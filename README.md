## Ecommerce_store

Объектно-ориентированная модель каталога интернет-магазина с поддержкой:

- категорий товаров

- различных типов продуктов

- создания заказов

- загрузки данных из JSON

- валидации бизнес-правил

- пользовательских исключений

Проект демонстрирует применение принципов ООП и построение доменной модели.

Проект создан как демонстрация навыков:

- построения объектной модели

- проектирования бизнес-логики

- соблюдения принципов ООП

- написания расширяемого и поддерживаемого кода

### Архитектура

#### Базовый уровень

- BaseProduct — абстрактный класс товара

- BaseCatalogObject — базовый класс объектов каталога

- Используются ABC, абстрактные методы и контрактная модель проектирования.

#### Модель товаров

**Product**

- инкапсуляция цены (private атрибут)

- property + setter с валидацией

- контроль изменения состояния

- перегрузка оператора + (подсчёт полной стоимости на складе)

**Smartphone и LawnGrass**

- наследование от Product

- расширение атрибутов

- типобезопасная перегрузка сложения

**Category**

- хранение списка товаров

- подсчет количества товаров

- вычисление средней цены

- классовые счетчики категорий и товаров

- защита от некорректных операций

**Order**

- создание заказа на основе товара

- проверка доступного количества

- расчет итоговой стоимости

- строгая бизнес-валидация

#### Дополнительно

- Кастомный итератор для перебора товаров

- Миксин для отладочного вывода объектов

- Иерархия пользовательских исключений

- Загрузка и преобразование данных из JSON

### Проект демонстрирует:

- Абстрактные классы (ABC)
